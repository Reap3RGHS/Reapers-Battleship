Download:

To download this project you must clone it using this command :

git clone https://github.com/Reap3RGHS/Reapers-Battleship

How to install:

Installation is very simple.

Don't download the zip/tar.gz file if you want to take all the new updates.

Install:

  • Open a terminal.

  • Type:

git clone https://github.com/Reap3RGHS/Reapers-Battleship

  • Go to the game directory:

cd Reapers-Battleship

  • Give to the game script root privileges:

chmod u+x reapers_battleship.sh

  • Run the game by typing:

./reapers_battleship.sh

You can open it with double click from the game folder.
