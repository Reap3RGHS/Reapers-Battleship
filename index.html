Download:

To download this project you must clone it using this command : git clone https://github.com/Reap3RGHS/Reapers-Battleship

How to install:

Installation is very simple.

If you want to take all the updates don't use the zip or the tar file but the git clone.

From Clone:

If you use git clone command to do this you must do the followings :

  1. Download it using >> git clone https://github.com/Reap3RGHS/Reapers-Battleship

  2. Go to game dir by typing this >> cd Reapers-Battleship

  3. Give it root privileges >> chmod u+x reapers_battleship

  4. Run it >> ./reapers_battleship

How to run it on all Linux distros :

To play on all linux distros you have to compile it. So follow this:

  1. Clone it by typing >> git clone https://github.com/Reap3RGHS/Reapers-Battleship

  2. Go to game dir >> cd Reapers-Battleship

  3. Compile using >> g++ -o reapers-battleship-working main.cpp

  4. Run it by typing >> ./reapers-battleship-working
